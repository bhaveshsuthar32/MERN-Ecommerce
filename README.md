# LEHARI Store

LEHARI Store is a sophisticated full-stack e-commerce platform developed using the MERN (MongoDB, Express.js, React.js, Node.js) stack. This comprehensive web application delivers a seamless and intuitive shopping experience for apparel across various demographics. Combining cutting-edge web technologies with user-centric design, LEHARI Store offers a responsive interface adaptable to multiple devices, enhanced by customizable viewing options. The platform features robust user authentication, advanced search capabilities, and secure payment processing integrated with Stripe. It provides categorized browsing for different clothing lines, personalized shopping lists, and dedicated sections for new arrivals. For administrators, a powerful dashboard facilitates efficient management of products, orders, user accounts, and customer feedback. LEHARI Store represents a state-of-the-art solution in online clothing retail, prioritizing user experience and operational efficiency.

## Features

### User Features

- User authentication (Login/Signup)
- Responsive web design with dark mode/light mode
- Sections for men, women, and kids clothing
- Sections for new arrivals and shopping list
- Advanced search functionality
- Payment gateway integration with Stripe
- User profile management (Update profile/password)
- Cart management (Add/Remove items, update quantities)
- Order management (View orders, order details)(not completed yet)

### Admin Features

As an admin, you have additional privileges and responsibilities for managing the e-commerce platform:

- **Dashboard Access**: Access to the admin dashboard with exclusive functionalities.
- **Update Order Status**: Ability to update the status of orders, such as processing, shipped, delivered, etc.
- **Delete Order**: Option to delete orders if necessary.
- **Add/Update Products**: Add new products to the inventory or update existing product details.
- **Delete User**: Delete user accounts if needed.
- **List Comment of Product**: View and manage product comment submitted by users.
